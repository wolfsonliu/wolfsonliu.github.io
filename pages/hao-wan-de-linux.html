看到一些好玩的 linux 程序, 记录下来.

Bash

Bash 下面的一些好玩的 linux 程序.

  • sl 小火车: 有的时候会把 ls 打错为 sl, 用该程序治愈以下. 会在 terminal 出现小火车, 还有多种选项.
  • asciiview: 可以使用符号在 terminal 上面显示位图.
  • cowsay: 在 terminal 显示一头说话的牛.
  • figlet: 可以根据参数给的字符画出来更大的字符画.
  • toilet: 和 figlet 类似, 也是根据给定的字符画更大的字符画.
  • fortune: fortune 是一个 python 程序, 可以随机地输出一个名言警句.
  • cmatrix: 用来在终端上产生类似于 The Matrix 电影中出现的那种绿色的字符雨.
  • sudo: 在 /etc/sudoers 的文件中的 Defaults 选项后面加上 "insults", 这样在每次输入密码错误后, sudo 会嘲笑用户.

桌面

  • xeyes: 打开一个 GUI 程序, 两只眼睛会盯着鼠标.
